## About

This project is a modern, award-worthy website featuring rich scroll-based animations, smooth page transitions, a product showcase, video trailer overlay, and a contact page. The design delivers a premium, cinematic feel across all devices.

## Features

- **Scroll-Based Animations** — Dynamic animations triggered by scrolling
- **Clip Path Transitions** — Unique geometric transitions using CSS clip-paths
- **3D Hover Effects** — Interactive 3D transformations on user interaction
- **Video Transitions** — Seamlessly integrated video elements for storytelling
- **Multi-Page Navigation** — Products, Buy Now, Trailer, and Contact overlay pages
- **Fully Responsive** — Flawless experience across all screen sizes

## Tech Stack

- React
- GSAP (GreenSock Animation Platform)
- Tailwind CSS

## Getting Started

### Prerequisites

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/) (v16 or above)
- npm

### Installation

```bash
git clone https://github.com/gourav/award-winning-website.git
cd award-winning-website
npm install
```

### Running Locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Deployment

### Vercel (Recommended)

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com) and import your repository
3. Vercel will auto-detect the framework as **Vite**
4. Configure:
   - **Framework Preset:** Vite
   - **Build Command:** `vite build`
   - **Output Directory:** `dist`
5. Click **Deploy**

### Netlify

1. Push your code to GitHub
2. Go to [netlify.com](https://netlify.com) and import your repository
3. Configure:
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`
4. Click **Deploy**

### Build for Production

```bash
npm run build
```

The optimized output will be in the `dist/` directory. You can preview it locally with:

```bash
npm run preview
```
